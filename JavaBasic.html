Java Hello World Program

public : Access modifier indicating visibility to other classes.
class: Keyword to declare a class.
HelloWorld: Name of the class.
public: Access modifier indicating visibility to other classes.
static: Indicates that the method belongs to the class itself, not instances of the class.
void: Return type indicating that the method does not return any value.
main: Name of the method, which serves as the entry point of the program.
String [] args : Parameter of the main method, representing command-line arguments.
System: Predefined class providing access to the system.
out: Public static member field of the System class, representing the standard output stream.
println: Method of the PrintStream class used to print a line of text to the output stream.
"Hello, World!": String literal to be printed to the standard output.
